 Python 多线程端口扫描器 (Port Scanner)
一个轻量、高效的多线程端口扫描工具，基于 Python 开发。支持自定义目标 IP、端口范围、线程数，并能快速识别端口开放状态。适合网络安全入门学习与内网资产探测。

核心功能
- 多线程并发扫描，速度快
- 支持自定义 IP、端口范围
- 精准识别端口开放/关闭/过滤状态
- 清晰的结果输出
- 纯 Python 标准库

---

 技术栈
- Python 3
- socket 网络编程
- threading 多线程

---

使用方法
1. 运行 main.py
2. 输入目标 IP
3. 输入端口范围（例如 1-1000）
4. 输入线程数
5. 等待扫描结果

---

 项目结构
- main.py        程序入口
- scanner.py     核心扫描逻辑
- config.py      配置文件
- utils.py       工具函数
